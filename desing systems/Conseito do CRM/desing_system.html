[05] MOTION & INTERACTION

Interactions & Transitions

All visual state updates utilize the custom Swiss curve `cubic-bezier(0.16, 1, 0.3, 1)`. Use the gallery below to preview and test the motion logic.

Scroll Entrance (reveal-up)

Elements start at `opacity: 0` and `transform: translateY(18px)`. They transition to default states once viewport intersections occur.

Border Scan Line (scan-line)

horizontal separator rules scale out from left-to-right (`scaleX(0) -> scaleX(1)`) on view entry.
